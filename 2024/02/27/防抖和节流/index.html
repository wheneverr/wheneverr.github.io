防抖和节流

前端

防抖

什么是防抖？

防抖(debounce)：触发高频事件后 n 秒内函数只会执行一次，如果 n 秒内高频事件再次被触发，则重新计算时间

如何实现防抖

/**
 * 非立即执行防抖函数
 * @param {Function} func 
 * @param {number} delay 
 * @returns 
 */
function debounce(func, delay) {
   let timeout
   return function () {
       const _this = this
       const args = [...arguments]
       if (timeout) {
           clearTimeout(timeout) // 如果之前计时器还存在则清除
       }
       timeout = setTimeout(() => {
           func.apply(_this, args)
       }, delay)
   }
}

节流

什么是节流？

节流(thorttle)：高频事件触发，但在 n 秒内只会执行一次，所以节流会稀释函数的执行频率

如何实现节流

  1. 使用时间戳实现
  • 时间戳版的功能函数触发是在时间段内开始的时候
function throttle(func, wait) {
    var previous = 0;
    return function() {
        let now = Date.now();
        let context = this;
        let args = arguments;
        if (now - previous > wait) {
            func.apply(context, args);
            previous = now;
        }
    }
}
content.onmousemove = throttle(count,1000);
  1. 使用计时器实现：
  • 第一次调用节流函数，还没有定时器，创建一个定时器，并在时间间隔结束时触发功能函数
function throttle(func, wait) {
     let timeout;
     return function() {
         let context = this;
         let args = arguments;
         if (!timeout) {
             timeout = setTimeout(() => {
                 timeout = null;
                 func.apply(context, args)
             }, wait)
         }
 
     }
}

为什么要防抖和节流？

  • 当函数绑定一些持续触发的事件如：resize、scroll、mousemove ，多次快速click等等，
  • 如果每次触发都要执行一次函数，会带来性能下降，资源请求太频繁等问题
