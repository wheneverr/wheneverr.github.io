css伪元素

CSS中的伪元素是用来选择元素的特定部分或位置，而不是整个元素。它们使用双冒号（::）表示，与伪类（pseudo-class）区别开来，伪类使用单冒号（:）。伪元素允许你通过CSS选择和样式化元素的特定部分，如元素的第一行、第一字母、或者生成的内容。
以下是一些常见的CSS伪元素以及它们的主要用途：

  1. **::before** 和**::after**： 这两个伪元素用于在元素的内容前面和后面插入生成的内容。它们经常用于添加装饰性的元素或图标。
.quote::before {
  content: '\201C'; /* 左引号 */
}

.quote::after {
  content: '\201D'; /* 右引号 */
}
  1. **::first-line** 和 **::first-letter**： 用于选择元素的第一行和第一字母，可用于样式化首行或首字母的样式。
p::first-line {
  font-weight: bold;
}

p::first-letter {
  font-size: 150%;
  color: #ff0000;
}
  1. **::selection**： 用于选择用户选中的文本部分，可以为选中文本应用特定的样式。
::selection {
  background-color: #ffee00;
  color: #000;
}
  1. **::placeholder**： 用于选择表单元素的占位符文本，可以样式化占位符文本的样式。
input::placeholder {
  color: #999;
  font-style: italic;
}
  1. **::marker**： 用于样式化列表项的标记部分（如
  2. 元素中的圆点或数字）。
li::marker {
  color: #ff0000;
  font-weight: bold;
}

这些是一些常见的伪元素，它们允许开发者更精细地控制和样式化文档的不同部分。使用伪元素可以增强页面的设计和交互效果。

2 平面转换

使用transform实现元素的平移旋转缩放等效果

  1. 位移效果-translate语法：
    transform: translate(水平移动距离, 垂直移动距离);取值（正负均可）像素值单位数值百分比（参照物为元素自身尺寸）注意：x轴正向为右，y轴正向为下如果需要过渡的效果，可以加上transition: all 0.5s;单方向位移如果translate只有一个参数，则默认为水平方向移动的距离单独设置某个方向移动的距离：translateX()或translateY()位移-绝对定位居中目标：使用translate快速实现绝对定位的元素居中效果positon: absolute;left: 50%；top: 50%;transform: translate(-50%, -50%)；
  2. 旋转效果语法：
    transform: rotate(角度);取值：角度一般为xxx deg技巧：取值正负都可，正为顺时针，负为逆时针旋转-转换原点（旋转中心）默认：盒子中心点语法： transform-origin: 原点水平位置 原点垂直位置;取值：方位（如left, right, top, bottom, center)，像素单位数值，百分比等。
    transform多重转换
    复合属性transform: translate() rotate();
  3. 缩放效果使用scale改变元素的缩放效果语法：transform: scale(x轴缩放倍数, y轴缩放倍数);技巧：一般情况下，只为缩放设置一个值，表示x轴和y轴等比例缩放transform: scale(缩放倍数)scale值大于1表示放大，scale值小于1表示缩小如果想要超出的部分隐藏，可以用overflow: hidden;
  4. 渐变效果（多个颜色逐渐改变，一般用于设置盒子的背景 ）例如背景颜色竖直方向渐变：background-color: linear-gradient(颜色1, 颜色2, …);如果想要半透明渐变：background-color: linear-gradient(transparent, 颜色2, …);

3 空间转换

目标：使用transform属性实现元素在空间内的位移，旋转和缩放等效果

  1. 空间位移语法：transform: translate3d(x, y, z);
    • transform: translateX();
    • transform: translateY();
    • transform: translateZ();
  2. 取值：（正负均可）像素单位数值，百分比透视：使用perspective实现透视效果属性（添加给父级）perspective: 值，取值为像素单位数值，一般在800-1200
  3. 空间旋转语法：
    • transform: rotateX(值)，沿着x轴转
    • transform: rotateY(值)，沿着y轴转
    • transform: rotateZ(值)，沿着z轴转，其实与平面旋转相同
  4. 该空间旋转效果也可结合透视属性使用，会更加真实拓展：rotate3d(x, y, z, 角度): 用来自定义旋转轴的位置及旋转的角度。
  5. 立体呈现语法：transform-style: perserve-3d;
  6. 缩放语法：transform: scale3d(值);
